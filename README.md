# segurancadepedreira
sistema completo de monitoramento automatizado para áreas de risco, projetado inicialmente para a zona de pedreira, mas aplicável a qualquer ambiente que exija vigilância de segurança. O sistema combina Visão Computacional, Inteligência Artificial (YOLOv8), geofencing, rastreamento por centróides, estimativa de distância e alertas sonoros/visuais
